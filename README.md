# LFR Solar Simulator Pro ☀️

An advanced, interactive performance analysis tool for Linear Fresnel Reflector (LFR) systems.

## 🚀 Live Simulation
This tool allows researchers and engineers to:
- Visualize solar ray tracing in real-time.
- Analyze Cosine and Shading efficiency across all sun angles.
- Adjust system parameters (Mirror Width, Receiver Height, Mirror Count) instantly.
- Download production-grade CSV reports for detailed analysis.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+).
- **Visualization**: Plotly.js (Charts and Ray Tracing).
- **Build Tool**: Vite.

## 📦 Local Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## 🌐 Deployment
The production-ready files are generated in the `dist/` folder. This folder can be hosted on:
- GitHub Pages
- Vercel
- Netlify
- Any static web server.
